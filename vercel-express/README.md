# Vercel Express v4

A simple Express v4 application deployed on Vercel.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to see the result.

## API Routes

- `GET /` - Returns a welcome message
- `GET /api/hello` - Returns "Hello World!"

## Deploy on Vercel

The easiest way to deploy this Express app is to use the [Vercel Platform](https://vercel.com).

```bash
vercel deploy
```
